# ngx-loader-ui

Reusable Angular Loader components library built with standalone components.

This project was generated using **Angular CLI v21.2.2**.

---

## Development server

Start a local development server:

```bash
ng serve
```

Open your browser and navigate to:

```
http://localhost:4200/
```

The application will automatically reload when you modify source files.

---

## Code scaffolding

Generate a new component:

```bash
ng generate component component-name
```

To see all available schematics:

```bash
ng generate --help
```

---

## Build

Build the project:

```bash
ng build
```

The build output will be stored in the `dist/` directory.

---

## Running unit tests

Run unit tests using Vitest:

```bash
ng test
```

---

## End-to-End Tests

Run e2e tests:

```bash
ng e2e
```

You can configure any testing framework you prefer.
